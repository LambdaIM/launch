# lambdacli version

## Introduction

Print the app version

## Usage

```
lambdacli version
```

Print all supported subcommands and flags:
```
lambdacli version --help
```

##Examples

```$xslt
lambdacli version --long

git commit: 524b33eab300caa23e4047fdfe497349d5aa9fd4
vendor hash: 
build tags: 
go version go1.12.6 darwin/amd64
```
