# lambdacli slashing

## Introduction

Slashing transactions subcommands

## Usage

```
lambdacli tx slashing <subcommand>
```

Print all supported subcommands and flags:
```
lambdacli tx slashing --help
```

## Available Commands

| Name                            | Description                                                   |
| --------------------------------| --------------------------------------------------------------|
| [unjail](unjail.md)                                   | free a validator or partner-validator                                      |
