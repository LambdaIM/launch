# lambdacli query pool markets

## Description

Query all markets 

## Usage
```
 lambdacli query pool markets [flags]
```

Print help messages:
```
lambdacli query pool markets -h
```

## Examples

Query all markets
```
lambdacli query pool markets 
```

After that, you will get all markets in chain
